# Pokémon Array Exercises 🚀

This project is a JavaScript exercise focused on mastering array manipulation. I have implemented various logic steps to transform, filter, and analyze a dataset of Pokémon names.

## Features & Implementation:

1.  **Alphabetical Sorting:** Organizes the list from A to Z using `.sort()`.
2.  **Reverse Order:** Flips the list using `.reverse()`.
3.  **Capitalization:** Uses `.forEach()` to print all names in uppercase.
4.  **B-Filter:** Filters the list to only include Pokémon starting with the letter "B".
5.  **Excluding "C":** Removes all Pokémon starting with "C" using `.filter()`.
6.  **Numbered Listing:** Prints the list in a "Number X - Name" format.
7.  **Dynamic Catching:** A function that adds new Pokémon to the list using `.push()`.
8.  **Strategic Insertion:** Adds specific Pokémon to the third position in the array using `.splice()`.
9.  **The Longest Name:** An algorithm that iterates through the array to find and return the name with the highest character count.

## Key Learnings:

*   Understanding the difference between **mutating** methods and those that return a new array (like `.filter()`).
*   Working with **Arrow Functions** (`=>`) and anonymous functions.
*   Mastering **Zero-based Indexing** (understanding that computers start counting at 0).
*   Using **Template Literals** (backticks \` \`) to combine strings and variables efficiently.
*   Implementing **Conditional Logic** (`if/else`) inside loops.
